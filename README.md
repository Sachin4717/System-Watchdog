# System Watchdog

System Watchdog is a Java-based system monitoring utility designed to detect zombie
and orphan processes and help optimize system performance.

## Architecture
- Backend: Process scanning & detection logic
- Frontend: Dashboard (CLI / JavaFX-ready)
- Resources: Styling & report output

## Features
- Zombie process detection
- Clean modular design
- Exportable report (CSV)
- Professional UI styling support
